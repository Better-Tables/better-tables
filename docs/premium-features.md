# Premium Features

## Bulk Actions
Perform actions on multiple selected rows simultaneously (delete, update, export, etc.)

## Export Options
Export table data in multiple formats (CSV, Excel, PDF) with customizable options

## Premium Adapters

### Enterprise Database Connectors
Connect to enterprise-grade databases (Oracle, DB2, SAP HANA, etc.)

### GraphQL Adapter with Subscriptions
Real-time data synchronization using GraphQL subscriptions

### Real-time Sync Adapters
Auto-sync table data with external sources in real-time

## Inline Table Editing
Edit data directly in cells with inline inputs, switches, and controls

## Calculated Columns
Automatically compute and display derived values (counts, sums, averages in relationships)

## PostGIS Distance Queries
Filter and sort by geographic distance using PostGIS spatial functions

## Advanced Filtering with AND/OR Conditions
Build complex filter queries with AND/OR logic in a dedicated drawer interface

## Data Import and Upload Management
Drag-and-drop CSV/Excel imports with validation, mapping, and batch processing

## Saved Views and Perspectives
Save filter/sort/column configurations and share them with team members

## Custom Aggregations and Grouping
Show totals, averages, and counts at column footers and group levels

## Row-Level Security
Control data visibility based on user permissions and roles

## Collaborative Editing
Multiple users editing simultaneously with real-time sync and conflict resolution

## Data Transformations
Pre-apply transformations to data (currency conversion, date formatting, calculations)

## AI-Powered Features
Smart column suggestions, anomaly detection, and auto-summarization

## Advanced Computed Fields
User-defined formulas and calculations across multiple columns

## Multi-Source Joins
Join data from different adapters, APIs, or data sources in a single table

## Data Validation and Quality
Cell-level validation rules with data quality indicators and completeness metrics

## Field-Level Security and Masking
PII masking, field-level permissions, and custom data obfuscation

## Table Templates
Save and reuse complete table configurations across projects
